# Final Project – CS3043

## Ethics of Artificial Intelligence in Assistive Technology

**Author: Anna Zadorenko**

---

## Project Description

This project is an interactive website that explores the ethical issues of artificial intelligence in assistive technology.

It focuses on how AI can help people with disabilities while also raising concerns about bias, privacy, and user-centered design.

The website includes:

* Explanation of benefits and risks of AI
* Interactive sections (tabs, slider, timeline)
* Real-world AI tools with links
* A short quiz to test understanding

## Purpose

The goal of this project is to raise awareness about ethical design in AI systems, especially those used by people with disabilities.

It emphasizes the importance of:

* Accessibility
* Fairness
* Privacy
* Real user involvement in design
